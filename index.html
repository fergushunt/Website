Some of my Technical Projects


Automated Testing Application


I created an application to test the processing of data uploaded by users to a hosted system. I extended a pre-existing framework using Selenium WebDriver to log onto the website and download the data to be tested. I used the Excel Interoperability library to move the data downloaded into preformatted test sheets, and to adjust the calculations performed in the sheets at runtime based on the data downloaded. This application allows developers to run tests faster and more often, this allows bugs to be caught more quickly after any changes are made to the system.

File Management System


I created an automated file management system which organised automatically generated files into folders and subfolders based on file metadata. A subsection of these files were then served to the customer. This system was built on a number of AWS servers. I created the program to organise the files on a pre-existing server. The organised files were passed to an S3 instance for long term storage and as a copy. To reduce the load on this server I set up a different EC2 instance to serve the files to the customer. This server used FTPS to allow the client to access the files. To avoid filling the server with out of date files, any files more than 30 days old were deleted.
